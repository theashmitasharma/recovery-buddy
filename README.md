A supportive web app for post-surgery recovery tracking and guidance.

![Recovery Buddy](logos/header_gentle_bloom.svg)

## Features

- **Personalized Recovery Tracking** - Track your healing journey day by day
- **Procedure-Specific Guidance** - Tailored information for 12+ common procedures
- **Daily Check-ins** - Monitor pain, swelling, bruising, and other symptoms
- **Emotional Support** - Mood tracking and supportive messaging
- **Recovery Milestones** - Visual countdown timers to key recovery stages
- **Daily Checklist** - Morning, afternoon, and evening recovery tasks
- **Surgeon Message Templates** - Pre-written messages for common communications
- **Recovery Journal** - Document your thoughts and progress
- **Photo Progress Tracker** - Visual timeline of your healing
- **Community Quotes** - Encouragement from others who've been there
- **Dark Mode** - Easy on the eyes for nighttime use
- **Celebration Animations** - Customizable celebrations when you complete tasks

## Tech Stack

- **Frontend**: Streamlit
- **Styling**: Custom CSS with luxury wellness spa aesthetic
- **Data**: Local JSON storage

## Local Development

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/recovery-buddy.git
cd recovery-buddy
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the app:
```bash
streamlit run app.py
```

4. Open http://localhost:8501 in your browser

## Deployment

This app is deployed on [Streamlit Cloud](https://streamlit.io/cloud).
